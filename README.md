注意这个爬虫不能爬取js生成的DOM也就是只能爬取手写的

# node-small-crawler

> this is a project that can download images from zhihu

# first

> ensure you have installed nodejs


# install

```
npm install

```

# run

> put url after crawler.js  that you want to get images from it

```

node crawler.js

```
